# travel
基于SpringBoot+vue旅游系统

## 一、项目简介
系统分为前台跟后台两个项目 前台： 展示旅游新闻、景区信息、美食信息、旅游线路、在线留言

后台管理员： 账号管理、地区管理、景点信息管理、地方美食管理、旅游线路管理、订单信息管理、新闻管理、系统管理 普通用户：旅游线路、订单信息
